3. 初始化局部变量和全局变量 
	3.1、初始化局部变量和全局变量 
		https://www.runoob.com/cplusplus/cpp-variable-scope.html
		
		
		当局部变量被定义时，系统不会对其初始化，您必须自行对其初始化。定义全局变量时，系统会自动初始化为下列值：

		数据类型	初始化默认值
		int			0
		char		'\0'
		float		0
		double		0
		pointer		NULL
		正确地初始化变量是一个良好的编程习惯，否则有时候程序可能会产生意想不到的结果。
