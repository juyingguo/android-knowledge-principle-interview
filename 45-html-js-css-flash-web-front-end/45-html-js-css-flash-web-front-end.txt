	3.9 HTML5和HTML5+是什么关系？
		https://www.zhihu.com/question/331007930/answer/1307302211
		虽然HTML5为移动端开发带来极大的便利，但是它仍然存在一些弊端，比如说在调用底层设备如摄像头、文件系统实现不佳等，业务方面如上传下载、支付、消息推送不够完善。为了弥补这些不足，HTML5+规范应运而生。除了解决了以上HTML5中的一些不足，HTML5+规范还提供了接近原生的渲染能力以及40万原生的API，使APP的体验效果大幅提升。
