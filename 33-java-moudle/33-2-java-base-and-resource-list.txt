	11.2 java泛型中使用T,E等的区别
		https://blog.csdn.net/chenlycly/article/details/25561029 
		
		E - Element (在集合中使用，因为集合中存放的是元素)
		 T - Type（Java 类）
		 K - Key（键）
		 V - Value（值）
		 N - Number（数值类型）
		？ -  表示不确定的java类型 
		：：这几个本质上没什么区别，只不过是一个约定好的代码，也就是说 使用大写字母A,B,C,D......X,Y,Z定义的，就都是泛型，把T换成A也一样，这里T只是名字上的意义而已
